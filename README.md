## 项目目的
通过本项目，了解并使用 Node.js，加深对 AngularJS、Bootstrap 的了解，整个搭建过程都将进行记录。
## 项目说明
此项目基于 **Node.js** 构建，列出一些使用到的东西：

 * **Node** 服务器端。
 * **AngularJS** 作为后台管理界面 MVVM 框架。
 * **Bootstrap 4** 作为后台管理界面 UI 框架。
 * **MongoDB** 作为数据库。
 * **Jasmine** 测试框架。
 * **Gulp** 自动化管理。
 * **RESTful** API接口规范。

## 使用
先把依赖的包安装下呗。
```javascript
npm install
```
再用进入项目根目录`node`运行。
```javascript
node app.js
```
运行下测试用例下也不错。
```javascript
npm test
```
